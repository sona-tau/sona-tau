    The ETCS Manifesto

    Sona Tau Estrada Rivera
    December 7, 2025

    Pre-prelude

    I use syntax similar to inference rules so check that out here, if you haven’t heard about it before:

    • Rule of Inference

    I do this because statements look very simple and clean like this, not because it is simpler to read.

    Prelude

    First, there are mathematical objects. The mathematical objects that the following axioms apply to are:

    There are sets.

    $$ \frac{\qquad\qquad}{X} $$

    For each set $A$ and $B$, there are some things called functions that go from $A$ to $B$.

    Now, let’s define some notation. The notation for a function $f$ from $A$ to $B$ can be any of the following, they are all equivalent: $$ f : A \to B \equiv A \overset{f}{\to} B. $$ Additionally, in the case above, $A$ is called the domain and $B$ is called the codomain.

    If we want to state two functions that go from A to B, we would write something like the following.

    $$ A \overset{f}{\to} B \land A \overset{g}{\to} B \equiv A \overset{f}{\underset{g}{\rightrightarrows}} B. $$

    Additionally, we can also write the arrows pointing from right to left (while preserving the order of the domain and codomain).

    $$ A \overset{f}{\to} B \equiv B \overset{f}{\leftarrow} A. $$

    When asserting that a function exists as a hypothesis, it is safe to assume that both the domain and codomain exist, without explicitly stating that they exist. So, in other words:

    $$ \frac{}{\quad A \overset{f}{\to} B \quad} $$

    Given two functions $f : A \to B$ and $g : B \to C$, there exists a function $g \circ f : A \to C$ called the composition of $f$ and $g$.

    $$ \frac{A \overset{f}{\to} B \overset{g}{\to} C}{A \overset{g \circ f}{\to} C} $$

    Let’s define some more notation. We don’t want to write the composition symbol every time we compose two functions. So, we will simply write both of them juxtaposed.

    $$ g \circ f \equiv g f. $$

    For each set $X$, a function $1_X : X \to X$ called the identity at $X$.

    $$ \frac{X}{\quad X \overset{1_X}{\to} X \quad} $$

    One more piece of notation, the identity at $X$ will also be written as $X$. This is safe to do because for each object, one can, and will, find an identity that is unique.

    $$ 1_X \equiv X. $$

    Axioms

    First

    There shall be associativity.

    $$ \frac{W \overset{f}{\to} X \overset{g}{\to} Y \overset{h}{\to} Z} {h \circ (g \circ f) = (h \circ g) \circ f} $$

    There shall be identities.

    $$ \frac{X \overset{f}{\to} Y \quad}{f \circ X = f \qquad Y \circ f = f} $$

    Intermezzo: Terminal objects

    Look at this predicate: $$ P(T): \forall X (\exists! X \to T). $$

    This predicate states that: given a $T$, for all $X$, there exists a unique function from $X$ to $T$. Any object $T$ that satisfies this predicate is called a terminal object. All sets that are terminal are equivalent (more on what this means later), so it is safe to pick one of them and give it a name. Call that object $1$.

    Second

    There shall be a terminal object.

    $$ \frac{}{\quad 1 \quad} $$

    Intermezzo: More notation

    Some notation.

    Some more notation, we want to define set membership as a function from that terminal object to any set. In naive set theory, singleton sets pick an element of the set. So in this case, a function from the terminal object “picks” something in the set.

    $$ x \in X \equiv \mathbf{1} \overset{x}{\to} X. $$

    Now, if we compose a function with a “picking” function, we can “pick” something in the codomain of the first function. This is exactly how evaluating a function at a value looks like. So, let’s define that as more notation.

    $$ \frac{X \overset{f}{\to} Y \qquad x \in X} {f(x) \equiv f \circ x} $$

    Third

    There shall be an empty set.

    $$ \frac{\qquad \qquad \qquad \qquad} {\exists! X (\not{\exists} x \in X)} $$

    Intermezzo: What does “empty” mean?

    You can think of this axiom as basically stating that the following is not only well-formed, but that it is also true.

    $$ \exists!X(\not\exists x \in X). $$

    The above predicate states that: there is a unique set X that does not have any members. A set that does not have any members must be empty.

    Fourth

    There shall be equality of functions.

    $$ \frac{X \overset{f}{\underset{g}{\rightrightarrows}} Y \qquad \forall x \in X (f(x) = g(x))} {f = g} $$

    Intermezzo: Products

    Say we have the sets $X$, $P$ and $Y$ with the functions $\pi_1 : P \to X$ and $\pi_2 : P \to Y$.

    We will say that a product of $X$ and $Y$ is $P$, if for all sets $I$ with functions $f_1 : I \to X$ and $f_2 : I \to Y$, then there exists a unique function $\langle f_1, f_2\rangle : I \to P$ such that $\pi_1 \circ \langle f_1, f_2\rangle = f_1$ and $\pi_2 \circ \langle f_1, f_2\rangle = f_2$.

    Using rules of inference, this looks like the following.

    $$ \frac{X \overset{\pi_1}{\leftarrow} P \overset{\pi_2}{\to} Y} {X \overset{f_1}{\leftarrow} I \overset{f_2}{\to} Y \implies \exists! I \overset{\langle f_1, f_2 \rangle}{\longrightarrow} P} \ (\pi_1 \circ \langle f_1, f_2\rangle = f_1 \land \pi_2 \circ \langle f_1, f_2\rangle = f_2) $$

    $$ X \times Y \equiv P. $$

    In this sense, the product of two sets is the Cartesian product as we would normally know it, but we also have two special functions. Namely, pi_1 and pi_2. The axiom does not state it but for all Cartesian products, these two functions are defined.

    Also, there is some more notation.

    $$ (X, Y) \equiv X \times Y. $$

    Fifth

    There shall be Cartesian products.

    $$ \frac{X \quad Y} {X \times Y} $$

    Intermezzo: Functions!

    Now we went to talk about the star of the show: functions. We would like to define a set of functions. For this, if we want to claim that a set $F$, is the set of functions from $X$ to $Y$, we will need one extra function $\epsilon : F \times X \to Y$. This function will take an element of $F$ (a function from $X$ to $Y$) and an element of $X$, and produce an element of $Y$. In other words:

    $$ \frac{f : X \to Y \quad x \in X} {\epsilon(f, x) \in Y} $$ It would be great if we had this function epsilon at our disposal (you will soon see why). So, how do we create such a function? If we find a way to “implement” it, we’re golden.

    A way to implement this function is as follows:

    $$ (f : X \to Y, x \in X) \mapsto f(x). $$

    In other words $\epsilon$ is a function that takes a function and an argument and applies that function to that argument. Now, we will use this to claim that $F$ really is the set we are looking for. If there is any other set $I$ with $\epsilon’ : I \times X \to Y$, then there must exist a unique function $q : I \to X$ such that $t \in I$ and $x \in X$ implies that $\epsilon’(t,x) = \epsilon(q(t), x)$. Using rules of inference, this looks like the following.

    $$ \frac{F \quad F \times X \overset{\epsilon}{\to} Y} {I \times X \overset{\epsilon’}{\to} Y} \ \rule[3pt]{15em}{0.05pt} \ \frac{\exists! q : I \to F \quad t \in I \land x \in X} {\epsilon’(t, x) = \epsilon(q(t), x)} $$

    $$ Y^X \equiv F. $$

    As it turns out, $|Y^X| = |Y|^{|X|}$. The amount of functions from $X$ to $Y$ is equal to $|Y|^{|X|}$, interesting!

    Sixth

    There shall be function sets.

    $$ \frac{X \quad Y} {Y^X} $$

    Intermezzo

    Not all functions have inverses. For example, the inverse of an element in the codomain of a function that isn’t surjective will not exist, and even if the element is in the image, if the function is not injective, the inverse could be any of a set of elements in its domain. For this, we must define the inverse of a function as a function that outputs a subset of the original function’s domain.

    For this, let $f : X \to Y, y \in Y$ and $j : A \to X$. The inverse of $f$ under $y$ is the set $A$ if for any $I$ and $q : I \to X$ the following is true: If $t \in I$ and $f(q(t)) = y$, then $\exists! q’ : I \to A$ such that $q = j \circ q’$.

    In other words, the following.

    $$ A \overset{j}{\to} X \overset{f}{\to} Y \quad y \in Y \ \rule[3pt]{15em}{0.05pt} \ I \overset{q}{\to} X \ \rule[3pt]{15em}{0.05pt} \ t \in I \ \rule[3pt]{15em}{0.05pt} \ f(q(t)) = y \ \rule[3pt]{15em}{0.05pt} \ \exists! I \overset{q’}{\to} A $$

    $$ \begin{align*} &InverseImage(\langle f : X \to Y, y \in Y, j : A \to X\rangle): \ &\qquad \forall I \land q : I \to X \ &\qquad \qquad (\forall t \in I (f(q(t)) = y) \implies \exists! \bar{q} : I \to A (q = j \circ \bar{q})). \end{align*} $$

    Now define the syntactic sugar:

    $$ \begin{align*} &A \land X \land Y, \ &y \in Y, \ &f : X \to Y, \ &j : A \to X, \ &\forall a \in A (f(j(a)) = y), \ &InverseImage(\langle f, y, j\rangle). \ \vdash; &f^{-1}(y) := A. \end{align*} $$

    Seventh

    There shall be inverse images.

    $$ \begin{align*} &X \land Y, \ &y \in Y, \ &f : X \to Y. \ \vdash; &\exists f^{-1} \circ y (\top). \end{align*} $$

    Intermezzo

    Fix a set called $2$:

    $$ \vdash; \mathbf{2}. $$

    These axioms imply that this set must have these two elements

    $$ \top \in \mathbf{2} \land \bot \in \mathbf{2}. $$

    Define the syntactic sugar:

    $$ \begin{align*} &A \land X, \ &j : A \to X, \ &\forall a \in A \land a’ \in A (j(a) = j(a’) \implies a = a’). \ \vdash; &j : A \hookrightarrow X := j. \end{align*} $$

    Define the predicate:

    $$ SubSet(\langle j : A \hookrightarrow X\rangle): \exists!\chi : X \to \mathbf{2} (\chi^{-1}(\top) = j). $$

    Then, define the syntactic sugar:

    $$ \begin{align*} &A \land X, \ &\exists j : A \hookrightarrow X. \ \vdash; &A \subseteq X := SubSet(\langle j \rangle). \end{align*} $$

    Eighth

    There shall be a subset classifier.

    $$ \begin{align*} \vdash; &\subseteq. \end{align*} $$

    Intermezzo

    Define the predicate:

    $$ \begin{align*} &NatSystem(\langle N, 0 \in N, s : N \to N\rangle): \ &\qquad \forall X \land a \in X \land r : X \to X \ &\qquad \qquad (\exists! x : N \to X \ &\qquad \qquad \qquad (x(0) = a \land \forall n \in N (x(s(n)) = r(x(n))))). \end{align*} $$

    Define the natural numbers as the following syntactic sugar:

    $$ \begin{align*} &N, \ &0 \in N, \ &s : N \to N, \ &NatSystem(\langle N, 0, s \rangle). \ \vdash; &\mathbb{N} := N. \end{align*} $$

    Ninth

    There shall be natural numbers

    $$ \vdash; \mathbb{N}. $$

    Intermezzo
